# IMDB Sentiment Analysis Streamlit App 

This project analyzes IMDB movie reviews and predicts their sentiment (positive or negative). It is built with **Streamlit** and uses **Logistic Regression + TF-IDF** for sentiment analysis.

## Features

- Analyze user-written reviews for sentiment
- Color-coded results: positive (green) / negative (red)
- Model performance metrics: Accuracy and F1 Score
- Optional: Confusion Matrix visualization
